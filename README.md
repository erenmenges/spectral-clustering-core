# Spectral Clustering from Scratch

A barebones implementation of spectral clustering using only NumPy. I built this to show why spectral methods work on non-convex clusters (rings, moons, spirals) where classic K-means falls apart.

Comes with an interactive Streamlit demo that walks through each step of the algorithm.

## How It Works

1. Compute pairwise distances via the Gram matrix
2. Build a mutual k-nearest-neighbor graph
3. Remove degree-≤1 outliers
4. Form the normalized graph Laplacian: L_sym = I − D^{−1/2} A D^{−1/2}
5. Eigendecompose L and embed points into the first few eigenvectors
6. Row-normalize the embedding, then run K-means in that space

The key insight is that the spectral embedding "unravels" non-convex structure so that simple K-means can separate the clusters.
This is a proof of concept and was only tested on the data generated by the generator script.

## Getting Started

```bash
pip install -r requirements.txt
```

You can run this code in two ways: by running it standalone or running it automatically via the demo.

### Generate datasets

```bash
python generate_plots.py --dataset rings
python generate_plots.py --dataset moons
python generate_plots.py --dataset spirals
```

Outputs `data_clean.csv` and `data_noisy.csv` (500 points each). You can also pass `--noise <float>` to control the noise level.

### Run spectral clustering

```bash
python cluster.py
```

Runs on `data_noisy.csv` by default and displays the k-NN graph and final clustering result.

### Run the demo

```bash
streamlit run demo/app.py
```

The demo has five steps: Input data, similarity graph, Laplacian eigendecomposition, spectral embedding, and final clustering. Also a side-by-side comparison against K-means on the raw data.

## Dependencies

- **NumPy** — all linear algebra and clustering logic
- **Matplotlib** — plotting
- **Streamlit** — demo app only

No scikit-learn or PyTorch. The whole point was to implement it myself.
